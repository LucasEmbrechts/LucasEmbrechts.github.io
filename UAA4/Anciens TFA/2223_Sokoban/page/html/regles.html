1. Vous devez trouver un moyen de déplacer toutes les caisses sans vous coincer sinonle joueur dois recommencer le niveau.

2. Le joueur peut se déplacer dans les quatre directions (haut, bas, gauche, droite) enutilisant les flèches du clavier.

3. Le joueur ne peut pousser qu’une seule caisse à la fois.

4. Vous ne pouvez pas tirer les caisses, vous devez les pousser.

5. Les caisses ne peuvent pas être poussées sur des obstacles tels que des murs ou d’autres caisses.

6. Le joueur ne peut pas sortir de la carte.

7. Le but du jeu est de placer toutes les caisses sur les croix en utilisant le moins de coups possible.

8. Une fois que toutes les caisses sont sur les croix, le niveau est terminé et le joueur peut passer au niveau suivant
