contexte :
tu as perdu tes objet retrouve les vite dans les différent monde
Principe :
Une partie se lance avec une image remplie d’objet cacher et une liste de mot il faut trouver toute les image de la liste le plus rapidement possible Une partie se termine quand toute les image seront retrouver Ils n’y a aucun moyen de perdre
Comment jouer:
• cliquez sur l’image d’un mot qui est sur la liste
• il y aura 3 maps différent de 25 d’images • Une partie = 1 map
• le chrono est par map
• lorsque je clique sur un mot et l’image est bonne l’image disparaît et le mot disparait à moitier
