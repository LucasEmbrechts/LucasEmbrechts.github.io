Changelog

Le changelog est une liste des plus gros changements pour chaque version du projet qui a été rendue.

Version 3 (finale) 26 mai 2017

Cette version est la dernière version du projet.

  • Changement des options du menu
  • Message quand le joueur interragi avec certains objets
  • Téléportation d'une map à une autre
  • Mise en place d'un site autour du jeu en lui-même

Version 2 5 mai 2017

  • Création d'une nouvelle map pour remplacer l'ancienne
  • Calibrage du mouvement du personnage
  • Corrections de quelques bugs

Version 1 31 mars 2017

  • Rendu d'une map
  • Déplacement du personnage sur une map
  • Menu avec différentes entrées sélectionnables
